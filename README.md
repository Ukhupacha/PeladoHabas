# Favapeel

Favapeel is an web app for fava control entries

## Requirements
* python 3.11 or later
### Backend
* fastapi
* uvicorn
* sqlalchemy
```bash
pip install -r requirements.txt
```

## Usage

```bash
python Backend/main.py
```
